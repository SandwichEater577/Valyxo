Why Valyxo?

No marketing fluff. Here's what Valyxo actually offers and why you might find it useful.

Being Honest

Valyxo is a student project. It's not competing with VS Code or professional IDEs. It's a learning project that became something usable.

Real Plugin System

Extend Valyxo with custom plugins. Unlike many student projects, this is a real plugin architecture:

  • Create plugins with plugin.json manifest
  • Register custom commands
  • Enable/disable plugins on the fly
  • Plugin templates for quick start

Built-in Package Manager

ValyxoScript has a package system with 8 built-in packages:

  • math, string, array, file utilities
  • http, json, time, random packages
  • Import syntax: import math
  • Install external packages

Git Integration

Full git commands built into the terminal. No switching to another terminal:

  • git status, add, commit, push, pull
  • Branch management and checkout
  • Clone repositories
  • View diffs and logs

Project Templates

Create complete projects with a single command:

  • Python, Node.js, React, Flask, Express
  • create react myapp scaffolds full app
  • Proper boilerplate and configs
  • Ready-to-run projects

9 Beautiful Themes

Full theme customization with real theme files:

  • Monokai, Dracula, Nord, Solarized, Gruvbox
  • Create and share your own themes
  • Export/import theme files
  • Full color customization

Enhanced ValyxoScript

ValyxoScript is now a real language with modern features:

  • Arrays with map, filter, reduce
  • Objects with dot notation
  • 50+ built-in functions
  • Import system for packages

25+ Code Snippets

Built-in snippets for Python, JavaScript, React, and more:

  • Prefix expansion (type trigger, get code)
  • Create custom snippets
  • Placeholder support
  • Search by name or tag

Customizable Keybindings

40+ default keybindings, all customizable:

  • Ctrl+C, Ctrl+L, arrow keys all work
  • Add your own shortcuts
  • Context-aware (editor vs prompt)
  • Export and share configs

Smart Auto-Complete

Tab completion that actually helps:

  • Commands, files, directories
  • Command history
  • Snippets and env variables
  • Fuzzy matching

Actual Working Terminal

The terminal isn't fake. It has real functionality:

  • File system operations (ls, cd, mkdir, cat)
  • Job management and process control
  • Man pages with built-in help
  • Command history with search

Fully Open Source

The entire codebase is on GitHub under MIT license:

  • Read and learn from the code
  • Fork and modify for your own projects
  • See how everything connects
  • No hidden functionality or tracking

Learning Project

Still a student project, but now with real features:

  • Python CLI with modular architecture
  • Electron desktop app
  • Modern HTML/CSS/JS website
  • Comprehensive documentation
