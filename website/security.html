Security

Valyxo Security Practices and Policies

Security Features

  • ✅ Password Hashing — bcrypt/argon2
  • ✅ Input Validation — Sanitization of all user inputs
  • ✅ HTTPS — Encrypted communication
  • ✅ Secure Secrets — Environment variables, no plaintext
  • ✅ SQL Injection Protection — Parameterized queries
  • ✅ CSRF Protection — Token validation
  • ✅ XSS Prevention — Output escaping

Reporting Security Issues

If you discover a security vulnerability, please email security@valyxo.dev with details.

Do not publish the vulnerability publicly until we have had time to address it.

Data Privacy

Your data is encrypted at rest and in transit. We never sell user data to third parties.

For more information, see our Privacy Policy.
