Overview

ValyxoScript is a lightweight, intuitive scripting language designed for simplicity and ease of use. It combines Python-like syntax with JavaScript flexibility.

Basic Syntax

set x = 10
set y = 20
set z = x + y
print z

if [z > 20] then [print "Greater!"] else [print "Smaller!"]

func add(a, b) {
  set result = a + b
  print result
}

Features

  • Variable assignment
  • Simple conditionals
  • Loop support (while, for)
  • Functions
  • Calculator mode
  • Print statements
  • String manipulation
