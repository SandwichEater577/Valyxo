API Reference

REST API documentation for Valyxo

Authentication

All API requests require authentication via JWT token.

Authorization: Bearer YOUR_JWT_TOKEN

Projects Endpoint

GET /api/projects

List all projects

Response: 200 OK

POST /api/projects

Create a new project

Response: 201 Created

GET /api/projects/:id

Get project details

Response: 200 OK

Scripts Endpoint

GET /api/scripts

List all scripts

Response: 200 OK

POST /api/scripts

Create a new script

Response: 201 Created

POST /api/scripts/:id/run

Execute a script

Response: 200 OK
