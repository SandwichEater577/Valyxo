## ğŸ› ï¸ Installation

### Prerequisites

- **Python 3.9+**
- **Node.js 18+** (for desktop app)
- **Rust** (for native modules, optional)
- **Go 1.21+** (for backend server, optional)

### Desktop App

```bash
cd Applications/Desktop
npm install
npm run tauri dev
```

### Go Server

```bash
cd Applications/ServerGo
go run cmd/server/main.go
```

### Build Native Modules

```bash
cd Libraries/Native
cargo build --release
```
